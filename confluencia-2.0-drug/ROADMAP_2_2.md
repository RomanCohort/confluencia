# Confluencia 2.2 两周冲刺计划（Drug）

## 目标
- 从“可运行原型”推进到“可验证、可复现、可对比”的研究版本。

## 第 1 周

### 1. 可信评估增强（已落地）
- 严格 train/val/test 切分。
- 5-fold 交叉验证统计（mean/std/95%CI）。
- 固定基线（Linear/RF/HGB）门槛判定。
- 失败样本 Top-N 导出。
- 外部独立测试集评估。
- 预测区间校准表（验证集残差校准测试覆盖率）。
- 泄漏审计（train/test 精确重叠率）。

### 2. 机制一致性（已落地）
- 多任务一致性约束：抑制不合理效益-风险组合。
- CTM 物理先验 + 残差混合：启发式锚点，降低参数漂移。

## 第 2 周

### 3. 发布级复现
- 每次训练自动保存：配置、数据哈希、环境依赖、核心指标。
- 统一实验报告模板（Markdown + CSV）。

### 4. 稳健性压力测试
- 小样本/缺失值/异常值鲁棒性回归测试。
- 按 group_id 分层评估稳定性。

### 5. 对外演示包
- 形成固定 Demo 数据集与对照结果。
- 形成答辩用指标看板截图脚本。

## 验收门槛
- Test RMSE 优于固定基线最佳模型。
- CV RMSE 95%CI 收敛（区间宽度可接受）。
- 泄漏重叠率接近 0。
- Smoke test 全绿。
